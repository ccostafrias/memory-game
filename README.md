## :joystick: How to play

There are three main configs:

|Config|Options|
|:---:|:---:|
|Memory theme|`numbers` or `icons`|
|Table size|`2x2`, `4x4` or `6x6`|
|Number of players|`1 - 4`|

### :one: Solo mode

> Find all the pairs of cards in the shortest time possible.
